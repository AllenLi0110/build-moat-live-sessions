QR Code Generator

dynamic link manager
FastAPI React + TS SQLite
Core Features
  • 🔗 Submit a long URL → get a short token + QR code image
  • 📱 Scan QR → 302 redirect — update target URL anytime without re-generating
  • 📊 Scan analytics: total count + per-day breakdown
  • ⏱ Expiration with live countdown, soft delete (410 Gone)
  • ⚡ In-memory cache-first redirect — DB only on cache miss
3 Design Decisions
token_gen.py SHA-256 + Base62 — collision-safe, URL-friendly token generation
url_validator.py Normalization + scheme check + blocklist — reject bad URLs at boundary
routes.py Cache → DB → 302 / 410 / 404 — one redirect path, three outcomes
